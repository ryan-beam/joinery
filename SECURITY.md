# Security

To report a vulnerability, use GitHub's Private Security Advisory feature:

https://github.com/ryan-beam/joinery/security/advisories/new

This routes the report directly to the maintainer without public disclosure.

## Status

Joinery is at v0.1.0 (pre-alpha). Once the framework stabilizes at v1.0, this file will document supported versions, response targets, and the full disclosure process. For now: report responsibly via the link above.
