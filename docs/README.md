# Documentation

Index for Joinery's docs.

## Contents

- [`architecture.md`](architecture.md) — 1-page architecture summary
- [`spec.md`](spec.md) — full design specification (~2000 lines, 18 sections)
- [`decisions/`](decisions/) — Architecture Decision Records (ADRs)

## Where to start

- **New to Joinery?** Read [`../README.md`](../README.md), then [`architecture.md`](architecture.md).
- **Want the design rationale?** Read [`spec.md`](spec.md) — long but exhaustive.
- **Curious about a specific decision?** [`decisions/`](decisions/) — chronological, each ADR has full context and alternatives.
